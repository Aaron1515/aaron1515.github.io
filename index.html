Workout Tracker

Connecting Trainers to Trainee

This app is created for fitness trainers and their clients. It's a way for trainer to issues workout, keep track of measurments, and for clients to view their work out for the weeks and days.

The app was built from the ground up using Rails. Opt to use a relational Postgres database, for quick write feature on low data input. Secured the app with an 128bit encryption for security login. Implemented D3.js graph for the measurements.

Want to check out the live version? Deoplyed on Heroku, or GitHub.
